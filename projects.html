Fun with animations

One of my earliest projects. It was created while I was first learning CSS. The goal of this project was to learn how to create animations on a webpage using CSS. It was created before I had a handle on responsive desgin so it is only moderately responsive. I tried to give it the feel of looking at an old black and green monitor.

Learning Bootstrap

This project was created with the goal of learning how to use the Bootstrap CSS framework. Although some of the Bootstrap styling is still visible, I changed most of the site to reflect my own styling.

E-Commerce Group Project

This was a group project created together with Emily Koceja, Javier Ramirez, John Taylor, and Karen Ngo. The goal was to create a responsive site with good UX/UI design that had functioning links. I focused primarily on creating and styling the product listing page. I also worked on the navigation bar, the animations on the home page, and helped my team as much as possible.
